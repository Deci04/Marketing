> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.
> Compagno di: `specs/2026-07-04-serata-multi-feature-design.md` §4 "Z", `plans/2026-07-04-orchestrazione-subagenti.md` §4-5, stile `plans/2026-07-04-fondamenta.md`.

**Goal:** Popolare la dashboard KPI **esistente** (`/kpi`, `getKpiData`) con dati reali presi da Zernio, tramite refresh **on-demand** (pulsante "Aggiorna dati", solo admin) — **niente cron**. Mappare: account/giorno → `Measurement`, demografiche → `AudienceSegment`, per-post → scalari su `Content` (match via `Content.externalId`) + append `MetricSnapshot`. Aggiungere il collegamento "Connetti account social" (OAuth lato Zernio) che salva `SocialAccount`. L'ingestione account/demografiche funziona **subito**; il per-post si accende man mano che W scrive `externalId` sui contenuti pubblicati.

**Architecture:** Il client `src/lib/zernio.ts` (già stub con `isConfigured`/`fetchAnalytics`/`publish`) diventa: (1) fetch autenticato verso `https://zernio.com/api/v1` con `Authorization: Bearer $ZERNIO_API_KEY`; (2) **mapper puri** raw-Zernio → record-KPI (unit-testabili con fixture JSON, **senza** rete); (3) `ingestAnalytics(workspaceId, analytics)` che scrive nel DB in modo **idempotente** (delete-then-create per giorno/dimensione, così refresh ripetuti non duplicano); (4) helper connect/callback OAuth. La server action `refreshKpiAction` (in `kpi/actions.ts`) è un **wrapper sottile**: guard admin → `fetchAnalytics` → `ingestAnalytics` → `revalidatePath("/kpi")`. Le route OAuth stanno sotto `src/app/api/integrations/zernio/**` (disgiunte da quelle Google del filone G). `publish()` resta lo stub del filone W (Z non lo tocca). Nessuna migration: schema già pronto da Fondamenta (`SocialAccount`, `Content.externalId`, `MetricSnapshot`).

**Tech Stack:** Next.js 16.2.9 (App Router custom — leggere `node_modules/next/dist/docs/`), Prisma 6 + Neon, next-auth v5, vitest 4 (node env, alias `@`→`src`), `fetch` nativo.

- **Date = mezzanotte UTC**: le date-giorno di Zernio (`"YYYY-MM-DD"`) diventano `new Date(`${ymd}T00:00:00.000Z`)`, mai fuso locale (coerenza con il resto del calendario/KPI).

Unità: `engagement_rate` come **frazione** (0..1, coerente con `engagementRate()` in `content.ts`); `non_follower_pct` come **percentuale** (0..100, coerente con `aggregatePerformance` che pesa `nonFollowerPct*reach`).

/** Fetch autenticato verso Zernio. Degrada solo se non configurato (chiamanti fanno la guard). */
export async function zernioFetch<T>(path: string, init?: RequestInit): Promise<T> {
  const res = await fetch(`${ZERNIO_BASE}${path}`, {
    ...init,
    headers: {
      Authorization: `Bearer ${key()}`,
      "content-type": "application/json",
      ...(init?.headers ?? {}),
    },
  });
  if (!res.ok) {
    const body = await res.text().catch(() => "");
    throw new Error(`Zernio ${res.status} ${path}: ${body.slice(0, 200)}`);
  }
  return (await res.json()) as T;
}

export function ymdToUtcMidnight(ymd: string): Date {
  return new Date(`${ymd}T00:00:00.000Z`);
}

export function platformToChannel(platform: string): Channel | null {
  return platform === "INSTAGRAM" || platform === "YOUTUBE" || platform === "TIKTOK"
    ? (platform as Channel)
    : null;
}

export type MeasurementUpsert = {
  date: Date;
  metric: "followers" | "engagement_rate" | "non_follower_pct";
  value: number;
  series: "Luca";
  channel: Channel | null;
};
export function mapAccountMeasurements(
  rows: ZernioAccountMetrics[], channel: Channel | null
): MeasurementUpsert[] {
  return rows.flatMap((r) => {
    const date = ymdToUtcMidnight(r.date);
    return [
      { date, metric: "followers", value: r.followers, series: "Luca", channel },
      { date, metric: "engagement_rate", value: r.engagementRate, series: "Luca", channel },
      { date, metric: "non_follower_pct", value: r.nonFollowerPct, series: "Luca", channel },
    ] as MeasurementUpsert[];
  });
}

export type AudienceSegmentUpsert = {
  date: Date; dimension: string; label: string; value: number; channel: Channel | null;
};
export function mapAudienceSegments(
  rows: ZernioDemographic[], channel: Channel | null, date: Date
): AudienceSegmentUpsert[] {
  return rows.map((r) => ({ date, dimension: r.dimension, label: r.label, value: r.value, channel }));
}

export type ContentPerfPatch = {
  views: number | null; reach: number | null; likes: number | null; commentsCount: number | null;
  saves: number | null; shares: number | null; followsGenerated: number | null; nonFollowerPct: number | null;
};
export function mapPostMetrics(p: ZernioPostMetrics): ContentPerfPatch {
  return {
    views: p.views, reach: p.reach, likes: p.likes, commentsCount: p.comments,
    saves: p.saves, shares: p.shares, followsGenerated: p.followsGenerated, nonFollowerPct: p.nonFollowerPct,
  };
}

- [ ] **Step 2: Implementare `ingestAnalytics` in `zernio.ts`**

Regole:
- **Measurement**: per ogni `MeasurementUpsert`, `deleteMany(scopedWhere(ws,{metric,series:"Luca",channel,date}))` poi `create` → idempotente per giorno/metrica/canale. (Nessun unique in schema → l'idempotenza è applicativa; delete-then-create in una `db.$transaction`.)
- **AudienceSegment**: la data del batch = `opts.snapshotDate ?? new Date()` **normalizzata a mezzanotte UTC** (`ymdToUtcMidnight(new Date().toISOString().slice(0,10))`). Per ogni segmento, `deleteMany(scopedWhere(ws,{dimension,label,channel,date}))` poi `create`. (Il read in `getKpiData` prende "first = latest ordered by date desc": una data-snapshot per refresh non intasa.)
- **Per-post**: per ogni `posts[]`, `db.content.findFirst(scopedWhere(ws,{externalId}))`; se assente → `postsMissing++` (opzione A: il match arriva quando W scrive l'externalId); se presente → `db.content.update({where:{id}, data: mapPostMetrics(p)})` **+** `db.metricSnapshot.create({data:{contentId, views,reach,nonFollowerPct,likes,saves,shares}})` (append storico) → `postsMatched++`.
- Usare `scopedWhere` **ovunque**; nessuna scrittura fuori dal `workspaceId`.

```ts
export type IngestSummary = { measurements: number; segments: number; postsMatched: number; postsMissing: number };

export async function ingestAnalytics(
  workspaceId: string,
  analytics: ZernioAnalytics,
  opts: { channel: Channel | null; snapshotDate?: Date }
): Promise<IngestSummary> {
  const measurements = mapAccountMeasurements(analytics.account, opts.channel);
  const snapDate = opts.snapshotDate ?? ymdToUtcMidnight(new Date().toISOString().slice(0, 10));
  const segments = mapAudienceSegments(analytics.demographics, opts.channel, snapDate);

  await db.$transaction([
    ...measurements.flatMap((m) => [
      db.measurement.deleteMany({
        where: scopedWhere(workspaceId, { metric: m.metric, series: "Luca", channel: m.channel, date: m.date }),
      }),
      db.measurement.create({ data: { ...m, workspaceId } }),
    ]),
    ...segments.flatMap((s) => [
      db.audienceSegment.deleteMany({
        where: scopedWhere(workspaceId, { dimension: s.dimension, label: s.label, channel: s.channel, date: s.date }),
      }),
      db.audienceSegment.create({ data: { ...s, workspaceId } }),
    ]),
  ]);

  let postsMatched = 0, postsMissing = 0;
  for (const p of analytics.posts) {
    const c = await db.content.findFirst({ where: scopedWhere(workspaceId, { externalId: p.externalId }), select: { id: true } });
    if (!c) { postsMissing++; continue; }
    const patch = mapPostMetrics(p);
    await db.content.update({ where: { id: c.id }, data: patch });
    await db.metricSnapshot.create({
      data: { contentId: c.id, views: patch.views, reach: patch.reach, nonFollowerPct: patch.nonFollowerPct, likes: patch.likes, saves: patch.saves, shares: patch.shares },
    });
    postsMatched++;
  }
  return { measurements: measurements.length, segments: segments.length, postsMatched, postsMissing };
}
```

```ts
export async function refreshKpiAction(): Promise<{ ok: boolean; error?: string; summary?: IngestSummary }> {
  const user = await currentUser();
  if (!user?.isAdmin) return { ok: false, error: "Non autorizzato" };
  const ctx = await currentContext();
  if (!ctx) return { ok: false, error: "Nessun workspace" };
  if (!isConfigured()) return { ok: false, error: "Zernio non configurato" };

  const accounts = await db.socialAccount.findMany({ where: scopedWhere(ctx.workspaceId) });
  if (accounts.length === 0) return { ok: false, error: "Nessun account social collegato" };

  let total: IngestSummary = { measurements: 0, segments: 0, postsMatched: 0, postsMissing: 0 };
  for (const acc of accounts) {
    try {
      const analytics = await fetchAnalytics({ profileId: acc.zernioAccountId, platform: acc.platform });
      const s = await ingestAnalytics(ctx.workspaceId, analytics, { channel: platformToChannel(acc.platform) });
      total = {
        measurements: total.measurements + s.measurements,
        segments: total.segments + s.segments,
        postsMatched: total.postsMatched + s.postsMatched,
        postsMissing: total.postsMissing + s.postsMissing,
      };
    } catch (e) {
      return { ok: false, error: `Zernio: ${(e as Error).message}` };
    }
  }
  revalidatePath("/kpi");
  return { ok: true, summary: total };
}
```

import { NextResponse, type NextRequest } from "next/server";
import { currentContext } from "@/lib/current";
import { saveSocialAccount, zernioFetch } from "@/lib/zernio";

export async function GET(request: NextRequest) {
  const ctx = await currentContext();
  if (!ctx) return NextResponse.redirect(new URL("/login", request.url));

  const cookie = request.cookies.get("zernio_oauth_state")?.value ?? "";
  const [, cookiePlatform] = cookie.split(":");
  const url = new URL(request.url);
  // Zernio ritorna l'id account collegato + handle (verificare i nomi param reali in Onda 0)
  const accountId = url.searchParams.get("account") ?? "";
  const handle = url.searchParams.get("handle");
  const platform = url.searchParams.get("platform") ?? cookiePlatform ?? "";

  if (!cookie || !accountId || !platform) {
    return NextResponse.redirect(new URL("/profilo?zernio=error", request.url));
  }
  await saveSocialAccount(ctx.workspaceId, { platform, zernioAccountId: accountId, handle });

  const res = NextResponse.redirect(new URL("/profilo?zernio=connected", request.url));
  res.cookies.delete("zernio_oauth_state");
  return res;
}
