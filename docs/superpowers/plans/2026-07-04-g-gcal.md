# Filone G — Google Calendar (sync bidirezionale, tutto) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Sync **bidirezionale** tra il board del tool e un **calendario Google condiviso dedicato**. USCITA: ogni mutazione dei chokepoint di `calendar.ts`/`content.ts` fa push/patch/delete su Google + upsert `GoogleCalendarLink`. ENTRATA: un webhook (push channel) innesca un pull incrementale (`syncToken`) che riconcilia `Content.publishAt` / date `Block` / `CalendarEvent`. Responsabile ↔ Google via tag titolo `[Luca]`/`[Matteo]` + colore + `extendedProperties.private.responsible`. Conflitti: last-write-wins via `etag`. Idempotenza: dedup su `googleEventId`.

**Architecture.** OAuth standalone (route in `src/app/api/integrations/google/**`) → token nel modello `Account` (provider `"google-calendar"`, ZERO migration). Client `src/lib/google-calendar.ts` centralizza: auth+refresh, `ensureCalendar`, encode/parse tag responsabile (funzioni pure), builder event-resource all-day, `syncItemOut`/`deleteItemOut` (USCITA idempotente), `pullChanges` (ENTRATA), watch-channel. I chokepoint restano nei loro file (`calendar.ts`, `content.ts`): aggiungono SOLO `import` + una chiamata fire-and-forget, così la logica di sync vive tutta in `google-calendar.ts`. `GoogleCalendarConfig`/`GoogleCalendarLink` esistono già (Fondamenta).

**Tech Stack:** Next.js 16.2.9 (App Router, Route Handlers), next-auth v5 (adapter Prisma → `Account`), Prisma 6/Neon, `googleapis` (nuova dep), vitest 4 (mock `googleapis`).

## Global Constraints

- **NIENTE commit finché non funziona tutto** (regola utente): i passi "commit" sono **checkpoint locali** — `git add` per raggruppare, ma **niente `git commit`/push** finché il filone non è verificato. Sostituire ogni "Commit" con "Checkpoint locale (staged, commit differito)".
- **Env senza virgolette** (gotcha Neon/token): `GOOGLE_CLIENT_ID`/`GOOGLE_CLIENT_SECRET` già configurate. Leggere via `process.env.X`; se mancano → `isConfigured()` è `false` e **tutto degrada in silenzio** (nessun crash sui chokepoint).
- **`scopedWhere(workspaceId, …)`** obbligatorio in ogni query di lib (`src/lib/workspace.ts`).
- **Prisma singleton** `src/lib/db.ts` (`import { db } from "@/lib/db"`); mai `new PrismaClient()`.
- **Next 16 custom**: già letto `node_modules/next/dist/docs/01-app/01-getting-started/15-route-handlers.md`. Fatti confermati: `export async function GET/POST(req: Request)`; Route Handlers **non cachati** di default (nessun `dynamic` necessario per POST/handler con side-effect); dynamic param via `RouteContext<'/…/[id]'>` con `await ctx.params` (qui non serve, nessun segmento dinamico); `route.ts` e `page.ts` **non** possono coesistere nello stesso segmento.
- **Webhook = secret/validazione header, MAI `currentContext()`** (non c'è sessione utente in una notifica push di Google). Il canale watch porta `X-Goog-Channel-ID`/`X-Goog-Resource-ID`/`X-Goog-Channel-Token`: si valida il `channelId` contro `GoogleCalendarConfig` e/o il token del canale.
- **Date calendario = all-day**: Google all-day usa `start.date`/`end.date` in formato `YYYY-MM-DD` (end **esclusivo**, = giorno+1). Le date del tool sono **mezzanotte UTC**; conversione via `toAllDayDate()`/`fromAllDayDate()` senza fuso locale.
- **Idempotenza**: ogni USCITA è un upsert su `GoogleCalendarLink (workspaceId, refType, refId)`; se esiste `googleEventId` → `patch`, altrimenti `insert`. Ogni ENTRATA dedup su `googleEventId`. Nessuna doppia scrittura.
- **Loop-guard**: una modifica arrivata DA Google (pull) **non** deve ri-triggerare una push verso Google. I chokepoint del tool sono l'unico punto di push; `pullChanges` scrive sul DB **senza** ri-chiamare `syncItemOut`.
- Modello dati/nomi campo **verbatim** dallo spec §3/§4 (`docs/superpowers/specs/2026-07-04-serata-multi-feature-design.md`).

```ts
import { google } from "googleapis";
import type { OAuth2Client } from "google-auth-library";
import { db } from "@/lib/db";

export function isConfigured(): boolean {
  return !!process.env.GOOGLE_CLIENT_ID && !!process.env.GOOGLE_CLIENT_SECRET;
}

export const GOOGLE_PROVIDER = "google-calendar";
export const CALENDAR_SCOPE = "https://www.googleapis.com/auth/calendar";

/** Costruisce un OAuth2 client con il redirect passato (per authorize/callback). */
export function newOAuthClient(redirectUri: string): OAuth2Client {
  return new google.auth.OAuth2(
    process.env.GOOGLE_CLIENT_ID,
    process.env.GOOGLE_CLIENT_SECRET,
    redirectUri
  );
}

/** Client autenticato per un workspace: token dall'Account del connettore,
 *  refresh + ripersistenza se scaduto. null se non configurato/non collegato. */
export async function getAuthClient(workspaceId: string): Promise<OAuth2Client | null> {
  if (!isConfigured()) return null;
  const cfg = await db.googleCalendarConfig.findUnique({ where: { workspaceId } });
  if (!cfg?.connectedByUserId) return null;
  const account = await db.account.findFirst({
    where: { userId: cfg.connectedByUserId, provider: GOOGLE_PROVIDER },
  });
  if (!account?.refresh_token) return null;

  const client = newOAuthClient(""); // redirect non serve per le chiamate API
  client.setCredentials({
    access_token: account.access_token ?? undefined,
    refresh_token: account.refresh_token,
    expiry_date: account.expires_at ? account.expires_at * 1000 : undefined,
  });

  const expiredSoon =
    !account.expires_at || account.expires_at * 1000 < Date.now() + 60_000;
  if (expiredSoon) {
    const { credentials } = await client.refreshAccessToken();
    client.setCredentials(credentials);
    await db.account.update({
      where: { id: account.id },
      data: {
        access_token: credentials.access_token ?? account.access_token,
        expires_at: credentials.expiry_date
          ? Math.floor(credentials.expiry_date / 1000)
          : account.expires_at,
      },
    });
  }
  return client;
}
```

```ts
import type { calendar_v3 } from "googleapis";
import type { BoardItemRef } from "@/lib/calendar";

export type Responsible = "LUCA" | "MATTEO" | null;
const TAG: Record<"LUCA" | "MATTEO", string> = { LUCA: "[Luca]", MATTEO: "[Matteo]" };
// Google colorId: 11 = tomato (Luca), 9 = blueberry (Matteo). Vedi Colors.get.
const COLOR: Record<"LUCA" | "MATTEO", string> = { LUCA: "11", MATTEO: "9" };

export function encodeTitle(label: string, responsible: Responsible): string {
  return responsible ? `${TAG[responsible]} ${label}` : label;
}

export function parseTitle(title: string): { label: string; responsible: Responsible } {
  const m = title.match(/^\[(Luca|Matteo)\]\s*(.*)$/s);
  if (!m) return { label: title, responsible: null };
  return { label: m[2], responsible: m[1] === "Luca" ? "LUCA" : "MATTEO" };
}

export function colorIdFor(responsible: Responsible): string | undefined {
  return responsible ? COLOR[responsible] : undefined;
}

export function toAllDayDate(d: Date): string {
  return d.toISOString().slice(0, 10);
}
export function fromAllDayDate(s: string): Date {
  return new Date(`${s}T00:00:00.000Z`);
}

export function buildEventResource(input: {
  label: string; date: Date; responsible: Responsible;
  refType: BoardItemRef; refId: string;
}): calendar_v3.Schema$Event {
  const next = new Date(input.date);
  next.setUTCDate(next.getUTCDate() + 1);
  return {
    summary: encodeTitle(input.label, input.responsible),
    start: { date: toAllDayDate(input.date) },
    end: { date: toAllDayDate(next) },
    colorId: colorIdFor(input.responsible),
    extendedProperties: {
      private: { responsible: input.responsible ?? "", refType: input.refType, refId: input.refId },
    },
  };
}
```

```ts
import { db } from "@/lib/db";
import { pullChanges } from "@/lib/google-calendar";

export async function POST(req: Request) {
  const channelId = req.headers.get("x-goog-channel-id");
  const state = req.headers.get("x-goog-resource-state");
  const token = req.headers.get("x-goog-channel-token");
  if (!channelId) return new Response("ok", { status: 200 }); // ignora non-Google
  const cfg = await db.googleCalendarConfig.findFirst({ where: { channelId } });
  if (!cfg) return new Response("ok", { status: 200 }); // canale sconosciuto → ignora
  if (process.env.GOOGLE_WEBHOOK_TOKEN && token !== process.env.GOOGLE_WEBHOOK_TOKEN)
    return new Response("ok", { status: 200 });
  if (state === "sync") return new Response("ok", { status: 200 }); // handshake iniziale
  await pullChanges(cfg.workspaceId).catch(() => {});
  return new Response("ok", { status: 200 });
}
```
